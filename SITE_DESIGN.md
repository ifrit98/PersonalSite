## 8) Palette and type direction

### Overall feel

Institutional + editorial + slightly symbolic

### Palette

* **Ink:** `#101826`
* **Bone:** `#F5F1E8`
* **Slate:** `#5E6875`
* **Brass:** `#A98752`
* **Oxblood accent:** `#4F2432`

### Usage

* Bone or off-white background by default
* Ink for headlines/body
* Slate for supporting UI text
* Brass sparingly for links, dividers, accents
* Oxblood only for rare highlights or section markers

### Type system

* **Headlines:** a serious serif
  Good direction: Canela / Tiempos / Newsreader
* **Body:** clean neutral sans
  Good direction: Inter / Suisse / IBM Plex Sans
* **Technical captions / metadata:** mono
  Good direction: IBM Plex Mono / JetBrains Mono

### Rules

* generous whitespace
* strong typographic hierarchy
* thin lines, not heavy boxes
* light motion only
* no glow effects, no neon, no hacker aesthetic

Second, the site should make three routes obvious from the first screen:

* **hire / collaborate**
* **read / subscribe**
* **evaluate credibility**
