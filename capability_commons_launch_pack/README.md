# Capability Commons

Capability Commons is an open knowledge system for practical literacy.

It maps concepts to skills, skills to projects, and projects to local deployment so people can move from explanation to competence and from competence to teach-forward transmission.

Founded and stewarded by **Jason St George**.  
Powered by **Agentic Data Lite**.

## What this project is for

The purpose of Capability Commons is to recover and organize practical knowledge that should be easier to access, easier to learn, and easier to pass on.

Initial domains include:

- water
- food
- shelter
- power
- repair
- gardening
- systems literacy
- community coordination

## Project surfaces

This project has three layers:

1. **Public learning surface**  
   Learning paths, modules, worksheets, reference sheets, and teach-forward packets.

2. **Knowledge architecture**  
   A graph that links concepts, skills, projects, modules, evidence, revisions, and local adaptations.

3. **Maintenance substrate**  
   Agentic Data Lite provides provenance, revision, retrieval, contradiction handling, and version-aware evidence assembly.
