# Capability Commons launch pack index

This folder contains the launch materials for the initial public release of Capability Commons.

## Files

- `README.md` — main public repository README
- `GOVERNANCE.md` — governance and stewardship model
- `BRAND.md` — brand and naming policy
- `homepage-copy.md` — exact homepage updates and short-form launch copy
- `capability-commons-landing-page.md` — dedicated `/capability-commons` page copy

## Recommended placement

- Put `README.md`, `GOVERNANCE.md`, and `BRAND.md` in the main public repo
- Apply `homepage-copy.md` to the home page and footer surfaces
- Use `capability-commons-landing-page.md` as the content source for `/capability-commons`

## Standard attribution line

> Capability Commons is an open capability commons for practical literacy, founded and stewarded by Jason St George, and powered by Agentic Data Lite.
